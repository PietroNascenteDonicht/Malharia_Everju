# Everju

Sistema de e-commerce desenvolvido com Laravel para gerenciamento e venda de produtos de uma malharia.

## Sobre o Projeto

Everju é uma plataforma de comércio eletrônico que permite aos clientes navegar por produtos, realizar pedidos e acompanhar compras, enquanto fornece aos administradores ferramentas para gerenciamento de produtos, categorias, pedidos e frete.
